# PC Guardian

Учебное desktop-приложение на C# WinForms для базового анализа безопасности компьютера.

## Сборка установщика

Из папки `Pz3`:

```powershell
.\build-installer.ps1
```

Готовый установщик:

```text
C:\Users\kavik\Desktop\Pz3\install.exe
```

Важно: удалятор исправлен. Он удаляет только файлы PC Guardian и больше не удаляет всю выбранную папку установки.
